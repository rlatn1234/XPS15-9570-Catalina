# DELL XPS15 9570 MacOS Catalina

MacOS Catalina EFI with **CLOVER / Opencore Bootloader**

Clover Bootloader supports MacOS Catalina 10.15.6(19G2021) 

OpenCore Bootloader supports MacOS BigSur 11.2.3(20D91)
